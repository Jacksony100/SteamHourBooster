# Roadmap

## Now

- Stabilize SaaS 2.0 local Docker flow.
- Keep the legacy Flask app preserved but inactive for new development.
- Finish public documentation and CI quality gates.
- Continue dashboard/admin polish after beta user feedback.

## Next

- Add typed shared API contracts in `packages/shared`.
- Add richer worker observability and queue dashboards.
- Add admin export/reporting for audit and billing events.
- Add production payment provider end-to-end tests with signed webhook fixtures.
- Capture hosted screenshot assets from the QA environment for README and release notes.

## Later

- Add organization/team accounts if needed.
- Add notification channels for session errors and subscription events.
- Add i18n message catalogs beyond the current language-ready structure.
- Add managed cloud deployment guides.
- Add structured OpenAPI client generation for the web app.

## Non-Goals

- Platform-circumvention features.
- Hidden automation.
- Credential logging or credential export.
- Mass account workflows.
- Promise of risk-free usage on third-party platforms.
